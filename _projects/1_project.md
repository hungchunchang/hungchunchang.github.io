The ECRobot project was conducted to investigate the effect of empathetic design on intelligent tutoring system. The system aims for assisting student to complete an linguistic puzzle, with llm-generated, empathetic feedback.

In this project, I developed the ITS system within 3 monthes alongside AI. The system includes three main components:

1. An customized iPad app developed in Swift, as the input of user's.
2. Interactive system with voice-user interface, allows partipants to discuss with the robot in realtime.
3. database management
